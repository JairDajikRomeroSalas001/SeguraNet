# **App Name**: Paucartambo Segura

## Core Features:

- Secure User Authentication: Implements user login functionality and automatically initializes two administrator accounts for initial system setup and management. (admin1:admin1, admin2:admin2)
- Case Registration Module: Provides a comprehensive form for entering new complaint details, including information about the complainant, incident description, specific location within Paucartambo, date, crime type, and additional relevant data.
- Dynamic Case Dashboard: Offers an overview of all registered cases, presenting a list with key information such as case number, current status, registration date, and direct access to full case details.
- Advanced Case Search & Filtering: Allows police officers to quickly locate specific complaints using multiple filtering criteria, such as case number, complainant name, date, complaint type, or case status.
- Case Status & Lifecycle Management: Enables the updating of case statuses (e.g., in progress, resolved, closed) and tracks changes to ensure a clear historical record for each complaint.
- AI-Powered Incident Classifier Tool: Analyzes the description of new incidents to automatically suggest relevant crime classifications or key tags, improving data accuracy and standardization during case registration.

## Style Guidelines:

- Primary color: A deep, authoritative blue (#36477D) chosen to evoke trust, security, and professionalism appropriate for a law enforcement system. This color represents the steady presence of order.
- Background color: A very light, almost imperceptibly cool gray (#F0F1F4), derived from the primary hue but highly desaturated and bright, providing a clean and understated canvas that keeps the interface clear and focused.
- Accent color: A vibrant blue-green (#2662D8) which is analogous to the primary color but with higher saturation and contrasting lightness, providing clear highlights for interactive elements and critical calls to action, maintaining a cohesive yet dynamic feel.
- All text: 'Inter', a grotesque sans-serif font known for its clarity and legibility across all screen sizes, ensuring readability for detailed case information and quick data scanning in an administrative context.
- Utilize clear, professional, and universally recognizable line icons for navigation, actions (e.g., add, edit, search), and status indicators, ensuring ease of use and quick comprehension.
- Implement a clean, organized, and tab-based layout for main sections, ensuring logical flow. Case listings should use a consistent card or table format, providing essential information at a glance with accessible detail views. Emphasize responsive design for optimal viewing across various devices.
- Incorporate subtle, functional animations for transitions between sections, form submissions, and feedback on user actions. These animations should be fast and non-disruptive, enhancing the user experience without drawing excessive attention.
